# ⌨️ TypingRush — 打字练习游戏

一个在浏览器中运行的打字速度练习工具，支持多种语言文章，提供计时和无时限两种练习模式。

## 功能

- **多语言文章**：支持中文、英文、编程代码三种类型的打字练习
- **两种练习模式**：计时模式（60 秒限时挑战）和无时限模式（不限时间，专注完成整篇文章）
- **排行榜系统**：本地记录每一次练习成绩，支持按照文章查看排行榜
- **GitHub Gist 云同步**：可选的排行榜云端同步，多设备共享数据（需要在设置中配置 GitHub Token）
- **深色 / 浅色主题**：点击页面右上角按钮切换
- **音效反馈**：正确输入和错误输入有不同音效提示
- **难度分级**：文章分为简单、中等、困难三个级别

## 如何使用

1. 打开 [TypingRush](https://hutiger9.github.io/typing-game/)（或直接在浏览器打开 `index.html`）
2. 选择一门语言，进入文章列表
3. 点击「开始练习」选择计时或无时限模式
4. 在输入框中对照上方文本逐字输入
5. 完成后可以保存成绩到本地排行榜

## ⚠️ 英文打字注意事项

**练习英文文章时，请务必先将输入法切换到英文状态**，否则会出现以下问题：

- 打字后光标不移动
- 正确输入不被识别
- 部分中文输入法候选框弹窗干扰输入

**操作要点：**
- 确保每次输入光标处字符时，输入框内无字符残留
- 建议将系统输入法切换为英文（美式键盘）后再开始英文打字练习
- 中文文章同理，请切换到中文输入法
